Interactive US regional map WordPress plugin with an easy to use admin panel interface.

== Description ==

The interactive US regional map plugin gives you an easy way to install and customize a professional looking USA map with 10 clickable Standard Federal Regions, then insert the shortcode in any page or post to display the map.

Are you looking for [Interactive US Map with Clickable States](http://codecanyon.net/item/interactive-us-map-wordpress-plugin/10359489?ref=clickmaps)?

Features:

*	Interactive US map with clickable Standard Federal Regions.
*	Does NOT require ANY coding knowledge.
*	Customize each region individually, so you can change colors, link, hover information for each region.
*	Easy to use admin panel interface.
*	Works perfectly on desktop along with mobile devices including Smartphones: iPhone, iPad, Tablets, etc.
*	Install and customize the map in minutes then embed the shortcode in any page or post to display the map.

You may be also interested in the [Interactive US Map with Clickable States](http://codecanyon.net/item/interactive-us-map-wordpress-plugin/10359489?ref=clickmaps)?

== Installation ==

1. Install the plugin .zip file through your WordPress 'Plugins' screen directly, or Upload the plugin files to the `/wp-content/plugins/us-regional-map` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Now, you will see a new menu button in the left side bar of your WordPress called 'US Regional Map', click it and customize your map then save the changes.
4. Use this shortcode `[us_regional_map]` to display the map in any page or post.

== Frequently Asked Questions ==

= Can I add an image in the hover popup? =

Yes, you can write any text and add images.

= Does this map requires Flash to work? =

No, it's SVG based and works perfectly on desktop along with mobile devices 'smartphone'.

= Is this map responsive? =

Yes.

= Can I split regions into states? =

No, but fortunately we already have an [Interactive US Map with Clickable States](http://codecanyon.net/item/interactive-us-map-wordpress-plugin/10359489?ref=clickmaps).

= Can I use the shortcode in different pages to display the same map? =

Yes.

= Can I create different map and use it in another page? =

No, but the [interactive US map by states](http://codecanyon.net/item/interactive-us-map-wordpress-plugin/10359489?ref=clickmaps) has that option and allows you to create a 2nd and 3rd map if needed and use them in different pages in your WordPress website.

= Can I link a region to any webpage? = 

Yes, also you can choose to open it in the same window or in a new window.

= Can I set a region as inactive? =

Yes, simply un-check the 'active' box beside that region to make it inactive.
