= Shortcode =

[reviews]
Copy & Paste this shortcode in your post or page to display the Google Business Reviews.

= Why this Plugin? =

This plugin allows you to display Google reviews on your WordPress pages and posts. Why is this important? Positive reviews on websites like Google help businesses gain additional exposure and further enhance their online credibility. A [recent study](http://marketingland.com/survey-customers-more-frustrated-by-how-long-it-takes-to-resolve-a-customer-service-issue-than-the-resolution-38756) conducted by Dimensional Research found that the buying decisions of 90% of consumers are influenced by positive online reviews. Additionally, a [2012 study published on Search Engine Land](http://searchengineland.com/study-72-of-consumers-trust-online-reviews-as-much-as-personal-recommendations-114152 "View the Article Online") found that 72% of consumers trust online reviews as much as personal recommendations. If you do the research it quickly becomes clear. Positive reviews add substantial credibility to any business and can lead to increased conversion rates and higher sales.

== Installation ==

**Automatic Install:**

1. Log into your WP-Admin
2. Navigate to 'Plugins' > 'Add New'
3. Search for "Google Reviews on Pages" and install it from there.

**Manual Install:**

1. Upload the `GoogleReviewsOnPages.zip` and extract it's contents to the `/wp-content/plugins/` directory or install via the WP plugins panel in your WordPress admin dashboard
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! You should now be able to use the widget.

== Frequently Asked Questions ==

= How to get my Google Place ID? =

1. Go to the Get Place ID(https://developers.google.com/places/place-id)

2. Enter location, copy the Place ID

= How to get my Google API key? =

1. Go to the Google API Console(https://console.developers.google.com/flows/enableapi?apiid=maps_backend,geocoding_backend,directions_backend,distance_matrix_backend,elevation_backend&keyType=CLIENT_SIDE&reusekey=true&pli=1).

2. Create or select a project.

3. Click Continue to enable the API and any related services..

4. On the Overview page search API “Google Places API Web Service” and Enable it.

5. On the Credentials page, Select API Key get a Browser key (and set the API Credentials)

6. Set "Key restriction" as None.

7. Note: If you have an existing Browser key, you may use that key.

Note* If Browser key does not work, Please generate and use Server key instead.
