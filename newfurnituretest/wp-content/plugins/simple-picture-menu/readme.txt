== Description ==

A menu bar placed on any side of the screen to create a picture based bar, for users to navigate through out your website.

Features<br>
-Simple Easy On/Off switch. (No Need to disable if desired)<br>
-Straight forward top, bottom, left, and right position option.<br>
-Fast small, medium, and large menu and icon resizing with the press of a button.<br>
-Quick Menu background color/transparent On/Off switch.<br>
-Made Simple option to scroll menu or stick in place On/Off switch<br>
-22 Pre-installed social media pictures with easy click and save controls<br>
-Easy title, picture location, and redirect link options for each and every image.<br>

== Installation ==

1. Download Simple Picture Menu from wordpress.org
2. Upload "Simple-Picture-Menu" to the  "/wp-content/plugins/" directory
3. Activate the plugin through the "Plugins" menu in WordPress
