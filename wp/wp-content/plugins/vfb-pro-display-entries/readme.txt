=== Visual Form Builder Pro - Display Entries ===
Contributors: mmuro
Requires at least: 3.5
Tested up to: 3.6.1
Stable tag: 1.5.4

Display your Visual Form Builder Pro entries in a fully sortable, filterable, and searchabe table.

== Release Notes ==

* Add notice that VFB Pro must also be installed and active first

** Version 1.5.3 - Aug 22, 2013**

* Update fields to include IDs in case of duplicate names (requires rebuilding shortcode)
