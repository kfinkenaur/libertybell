== Instructions ==
We recommended that you use GMO Showtime and GMO Font Agent plugins for maximum result.
-Install and activate GMO Showtime slider plugin to enable homepage slider.  
-Install and activate GMO Font Agent plugin to enable custom web fonts. 
-To upload your own logo, go to Appearance > de naani > Logo Image. 
-To change title position, go to Appearance > Site Title & Tagline > Title Position. 
You can download plugins from following links. 
http://wordpress.org/plugins/gmo-showtime/
http://wordpress.org/plugins/gmo-font-agent/
